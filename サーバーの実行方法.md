・Linuxサーバーにログイン後このコマンド3つを実行
cd GreedyProject　　　　　　　　　　##初回ログイン時のみ
source .venv/bin/activate　　　　　##初回ログイン時のみ
python manage.py runserver 0.0.0.0:8000

・自分のＰＣで開発したときこのコマンドでサーバーにアップロードできる
git add .
git commit -m "ここには好きなコメントを入れる"
git push origin main

・変更した場合はこのコマンドでサーバーに変更をダウンロードして実行
git pull origin main
python manage.py runserver 0.0.0.0:8000
